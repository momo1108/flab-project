# Watcha 클론 프로젝트 요구사항 정의서 (PRD)

## 1. 프로젝트 개요

### 목표

TMDB(The Movie Database) API를 활용하여 영화 정보 조회 및 검색 서비스 구현 (Watcha 클론)

### 범위

- 메인 페이지: 인기 영화, 상위 평점 영화, 개봉예정 영화 (3개 섹션)
- 검색 페이지: 영화 검색 및 결과 조회
- 상세 페이지: 선택 영화의 상세 정보 표시

### 배포

- CloudFront + S3를 통한 정적 호스팅

---

## 2. 기능 요구사항 (Functional Requirements)

### FR-1: 메인 페이지

- **FR-1.1**: 인기 영화 섹션 (상위 20개)
- **FR-1.2**: 상위 평점 영화 섹션 (상위 20개)
- **FR-1.3**: 개봉예정 영화 섹션 (상위 20개)
- **FR-1.4**: 각 영화카드 클릭 시 상세페이지로 이동
- **FR-1.5**: 무한 스크롤 또는 "더보기" 버튼 (선택)

### FR-2: 검색 페이지

- **FR-2.1**: 검색바에서 영화명 입력 가능
- **FR-2.2**: 입력 후 자동 검색 (Debounce: 300ms)
- **FR-2.3**: 검색 결과 그리드 표시
- **FR-2.4**: 페이지네이션 지원 (10개 항목/페이지)
- **FR-2.5**: 검색 결과 없음 상태 표시
- **FR-2.6**: URL에 쿼리 파라미터 반영 (`?query=keyword&page=1`)

### FR-3: 상세 페이지

- **FR-3.1**: 영화 기본정보 표시
  - 포스터 이미지
  - 배경 이미지 (배너)
  - 제목
  - 평점 (별 또는 수치)
  - 개봉일
  - 장르
  - 러닝타임
- **FR-3.2**: 영화 상세정보 표시
  - 줄거리 (Overview)
  - 제작사
  - 태그라인 (선택)
- **FR-3.3**: URL 파라미터로 영화 ID 관리 (`/movie/:id`)
- **FR-3.4**: 뒤로가기 버튼 또는 브라우저 뒤로 기능

### FR-4: 공통 기능

- **FR-4.1**: 헤더 네비게이션 (로고, 메인, 검색)
- **FR-4.2**: 로딩 상태 표시
- **FR-4.3**: 에러 상태 처리
- **FR-4.4**: 이미지 로딩 실패 시 플레이스홀더

---

## 3. 비기능 요구사항 (Non-Functional Requirements)

### NF-1: 성능

- **NF-1.1**: 페이지 초기 로드 시간 < 3초
- **NF-1.2**: 이미지 로딩 최적화 (TMDB 이미지 크기 제한)
- **NF-1.3**: API 응답 캐싱 (Tanstack Query)
  - 인기/평점/개봉예정: 5분 staleTime
  - 상세페이지: 10분 staleTime
  - 검색결과: 3분 staleTime

### NF-2: 사용성

- **NF-2.1**: 반응형 디자인 (모바일/태블릿/데스크톱)
- **NF-2.2**: 마우스 호버 효과 (카드)
- **NF-2.3**: 모바일 터치 친화적 UI

### NF-3: 신뢰성

- **NF-3.1**: API 에러 재시도 (최대 1회)
- **NF-3.2**: 네트워크 오류 시 사용자 친화적 메시지

### NF-4: 보안

- **NF-4.1**: API Key는 환경변수로 관리
- **NF-4.2**: HTTPS 전송 (CloudFront 기본)

### NF-5: 확장성

- **NF-5.1**: CSS Modules로 스타일 격리
- **NF-5.2**: 컴포넌트 재사용 가능하도록 설계

## 4. 데이터 요구사항

### 외부 API

- **TMDB API v3**
  - Endpoint: `https://api.themoviedb.org/3`
  - 인증: `api_key` 쿼리 파라미터

### 필수 API 엔드포인트

| 기능      | 엔드포인트         | 용도       |
| --------- | ------------------ | ---------- |
| 인기 영화 | `/movie/popular`   | 메인페이지 |
| 상위 평점 | `/movie/top_rated` | 메인페이지 |
| 개봉예정  | `/movie/upcoming`  | 메인페이지 |
| 영화 검색 | `/search/movie`    | 검색페이지 |
| 상세정보  | `/movie/{id}`      | 상세페이지 |

### 데이터 구조

```typescript
// 영화 목록 항목
{
  id: number;
  title: string;
  poster_path: string;
  backdrop_path: string;
  overview: string;
  vote_average: number;
  release_date: string;
  genre_ids: number[];
}

// 영화 상세 정보
{
  ...위와 동일...
  runtime: number;
  genres: Array<{ id: number; name: string }>;
  tagline: string;
  production_companies: Array<{ name: string }>;
}
```

## 5. 제약사항 (Constraints)

- **개발 범위**: 메인/검색/상세 페이지만 (회원가입, 찜하기, 평점 등 제외)
- **TMDB API 제한**: 무료 버전 사용 (초당 요청 제한 고려)
- **배포**: CloudFront + S3 정적 호스팅
- **브라우저 지원**: 최신 2개 버전 (Chrome, Firefox, Safari, Edge)
