바로 실행 가능한 운영 팁 3가지

1. 티켓마다 “Template 1개 + 해당 체크리스트”를 한 쌍으로 Claude Code에 투입
2. 응답 마지막에 항상 “체크리스트 pass/fail 표”를 출력하도록 고정
3. fail 항목은 다음 티켓 시작 전에 우선 수정하도록 워크플로우 고정
